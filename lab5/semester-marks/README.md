# Semester Marks Management System

## Objectives
- Build a responsive frontend using React to enter/view semester marks
- Create a RESTful backend with Spring Boot to calculate, store and retrieve results
- Persist results in MySQL (alternatively MongoDB/Oracle)
- Compute subject totals and semester percentage using MSE (30) + ESE (70) weightage
- Display grade and pass/fail status and allow download/print of result
- Learn full-stack integration: React → REST → Spring Boot → Database
- Make UI responsive using CSS and validate inputs

## Tech Stack
- Frontend: React
- Backend: Spring Boot
- Database: MySQL

## Setup Instructions

### Backend (Spring Boot)
```bash
cd backend
mvn spring-boot:run
```

### Frontend (React)
```bash
cd frontend
npm install
npm start
```
